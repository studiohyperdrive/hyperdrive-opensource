Hyperdrive
Open Source

Hyperdrive Open Source is a mono-repo providing several of the Typescript based packages created and maintained by the Studio Hyperdrive team.

Utils

Several independent utilities to facilitate common use-cases for users and developers.

Forms

Multiple utilities for complex form use-cases.

Store

Several utils and abstractions to improve the Redux workflow

Table

A quick and easy template based table builder using the Angular CDK Table.

I18n

A lazy-loaded modular approach to translations.

Cookies

A quick and easy wrapper for CookieConsent V3.

Layout

A collection of Angular components related to layout.

Tour

The tools to build a guided walkthrough of one or more pages.

At Studio Hyperdrive, we craft robust digital solutions with expertise in JavaScript and Angular since 2018. This passion led to Hyperdrive Open Source (HDOSS), an initiative launched in 2021 that has grown into an active ecosystem.

We’ve created 16 packages to date, for a.o. Angular, RXJS, and Node.js. HDOSS tools have been widely implemented and received international interest, thus empowering developers around the globe to streamline workflows and build better applications.

Why Hyperdrive Open Source?

Collaboration with the community

Engaging with the global developer community helps us share knowledge, learn, and create better tools together.

Efficiency and simplicity

Our packages eliminate repetitive tasks, letting you focus on real problems with a reliable foundation that simplifies and speeds up development.

Proven building blocks

Our tools are modular, proven building blocks that reduce bugs, ease production stress, and free up time for innovation, all while meeting the latest WCAG standards

Built for developers

Hyperdrive Open Source is built by developers, for developers. Our tools are designed to save you time, reduce frustration, and help you ship better code faster. Whether you’re looking for UI components, utilities, or advanced functionality for e.g. Angular applications, our packages have something to offer. And if not, stay tuned. We’re working every day on expanding them.
